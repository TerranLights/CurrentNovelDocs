# Phase 4 — Story & Chapters

**Prerequisite:** Phase 2 (Characters) and Phase 3 (Supporting Cast, if applicable) complete.

**What this phase does:** Completes the full plot structure and chapter/stanza breakdown — the point at which everything needed to actually draft the manuscript in full exists. This is "what do these people do?" — built on who they are (Phase 2–3), and built *before* what it all means (Phase 5).

---

## 4.0 — Standalone vs. Series: Which Scale(s) Apply

Before mapping beats, decide how many levels of structure this project needs:

- **Standalone novel:** one beat-structure mapping, covering the whole book. Sections 4.1–4.4 below apply directly.
- **Series:** the beat structure can be applied at more than one scale at once — a series-level arc spanning all books, *and* each individual book having its own complete sub-arc within it. This isn't a special case to improvise; it's the same nested approach the shared-universe timeline itself uses (the First Interwar Period and Second Interwar Period each get their own independent beat-by-beat mapping at a centuries-long scale, while individual scenes get the small-scale per-stanza mini-arc). Decide explicitly:
  - What is the series-level Opening Image / Midpoint / All Is Lost / etc., spanning the whole series?
  - What is *this* book's own Opening Image / Midpoint / All Is Lost, as its own complete arc?
  - How does this book's own Break into Three / Finale relate to the series' larger structure — is this book a full Act within the series' beats, or does it have its own three-act shape nested inside one series-level beat?

Note the choice made here at the top of `01-Completion-Matrix.md`'s Story & Chapters section, since it changes what "complete" means for the rest of this phase.

## 4.1 — Main Beat Structure

Map this novel's (or this book's) plot against the fused beat structure in `structure/main story/Save the Cat.md` and `reference/story-structure-definitions.md`:
- Every beat identified with an actual story event (not left as a blank placeholder)
- Placement-flexible beats (Self-Revelation, Moral Decision, Q-Factor, the three Campbell additions) assigned a specific location, not left floating
- Each core-cast character's own arc (`structure/character arcs/`) checked against the main beat structure for consistency — a character's Midpoint should land near the story's own Midpoint, etc.

## 4.2 — Chapter/Stanza Breakdown

Using `structure/main story/stanza scene and chapter breakdowns/outline-template.md`:
- One file per chapter (or combined file, if the hand-written source genuinely doesn't split cleanly — see `02-Blocking-Decisions.md` for how to handle an ambiguous split)
- Every stanza/scene has a Summary and all three Hero's-Journey mini-beat columns filled in
- Scene breaks marked
- Cut/blank stanzas flagged, not silently dropped

## 4.3 — Subplots

For each subplot (B-Story and beyond):
- Where it starts, where it converges with the A-plot (per the B-Story beat definition), and what it pays off
- Checked against the main beat structure for timing

## 4.4 — Full Draft

- Prose draft written from the completed chapter/stanza breakdowns
- Track progress per-chapter, not just as one binary "done"

---

## Phase 4 Output Checklist

- [ ] Standalone vs. series scale decided (4.0)
- [ ] Main beat structure fully mapped at every applicable scale (no blank/TBD beats remaining)
- [ ] All chapters broken down into stanza/scene tables
- [ ] All subplots mapped against the main beat structure
- [ ] Full prose draft complete

**When all items are checked: this novel's outline work is functionally done. Phase 5 (Theme) can now be identified with confidence, and only drafting/revision (Phase 7) remains.**

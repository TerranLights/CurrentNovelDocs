# Phase 6 — World Depth

**Prerequisite:** Phase 1 complete.

**Runs alongside:** Phases 2 through 5 — this phase doesn't block or get blocked by the character/plot/theme work, aside from needing Phase 1's foundations settled first.

**What this phase does:** Fills in setting detail beyond what's strictly load-bearing for the plot — the texture that makes the world feel lived-in, not just a backdrop for the main cast.

---

## 6.1 — Secondary Locations

Any location that appears on-page but isn't the primary setting — enough detail that a scene set there feels specific, not generic.

## 6.2 — Historical/Cultural Texture

- How does this story's setting connect to the shared-universe timeline beyond the bare minimum needed for Phase 1's consistency check?
- Local culture, customs, or history specific to this story's setting that add texture without being plot-critical.

## 6.3 — Consistency Pass

Before calling this phase done, check every world-building fact introduced here against:
- The shared universe timeline/reference material
- This novel's own Phase 1 foundations
- Any other novel in this line that shares the same setting/era, if applicable

---

## Phase 6 Output Checklist

- [ ] Secondary locations detailed
- [ ] Historical/cultural texture written
- [ ] Consistency pass complete against shared-universe material

**This phase has no strict completion gate on other phases — work on it whenever useful, but don't let it delay Phase 2/3/4.**

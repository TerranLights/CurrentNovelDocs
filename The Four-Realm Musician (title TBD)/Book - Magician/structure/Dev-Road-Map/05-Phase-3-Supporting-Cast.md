# Phase 3 — Supporting Cast

**Prerequisite:** Phase 1 complete.

**Runs alongside:** Phase 2 (Characters) — this isn't a later step; group/organization "wants" are decided at the same time as core-cast character work, since both are part of the same "who are these people, and what is this place?" foundation.

**What this phase does:** Designs any organizations/groups the plot depends on. Not every novel needs this phase — skip it if there's no group beyond the core cast that matters to the plot.

---

## 3.1 — Organizations & Groups

When this phase applies (a band, a business, a family beyond the core trio, an institution the plot pushes against), for each group:

- What does the group want, and what do they need that only their opposition (or the protagonist) can provide? (Avoid designing groups as simple obstacles — per the Fake Ally concept in `story-structure-definitions.md`, opposition that acts from its own genuine, internally consistent motivation is stronger than opposition that exists just to block the protagonist.)
- How does the group relate to the core cast — ally, obstacle, or something that shifts over the story?

This is decided now, at the same time as character work — not deferred to a later "theme" pass. What a group *wants* is part of who they are, not part of what the story *means*.

---

## Phase 3 Output Checklist

- [ ] Every plot-relevant organization/group designed (or explicitly noted as not applicable)

**When all items are checked (or this phase is marked not applicable): Phase 4 can begin (alongside Phase 2, which should already be underway).**

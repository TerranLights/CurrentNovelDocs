# Phase 7 — Long-Term

**Prerequisite:** Phase 4, Phase 5, and Phase 6 complete.

**What this phase does:** Everything that only makes sense once the novel itself is fully outlined (or drafted) — sequel planning, deliberately deferred world-building, and the revision pass.

---

## 7.1 — Sequel / Series Planning

If this novel is part of a series:
- What does the next book need from this one (setups, unresolved threads deliberately left open)?
- Cross-check against the shared-universe timeline so the series' internal chronology stays consistent.

## 7.2 — Deferred World-Building

Anything flagged "TBD" or deliberately deferred in earlier phases, revisited now that the full story is settled.

## 7.3 — Revision / Editing Pass

- Continuity check across the full manuscript
- Consistency check against the shared-universe reference material one more time, now that the full draft exists
- Line-level prose revision

---

## Phase 7 Output Checklist

- [ ] Sequel/series setups identified and confirmed intentional (if applicable)
- [ ] All deferred items from earlier phases resolved or explicitly re-deferred with reasoning
- [ ] Full revision pass complete

**When all items are checked: this novel is ready for submission/publication.**
